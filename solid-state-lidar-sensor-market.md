<p><ul><li>MEMS Based Scanning</li><li>Phase Array</li><li>Non-Scanning Flash</li></ul></p>
<p><p>The Solid State LiDAR Sensor market features three main types: MEMS-Based Scanning, Phase Array, and Non-Scanning Flash. MEMS-Based Scanning utilizes micro-electromechanical systems for precision scanning, providing high-resolution 3D mapping. Phase Array LiDAR employs multiple laser beams simultaneously, allowing for rapid data acquisition with minimal moving parts. Non-Scanning Flash LiDAR emits a broad light pulse, capturing entire scenes in a single snapshot, making it suitable for fast-moving applications. Each type offers distinct advantages depending on the use case.</p></p>
